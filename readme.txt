== Description ==

With Admin Setting you can easily customize the WordPress admin menu and toolbar and customize the admin and login interfaces
Admin Setting provides an awesome and simple interface to customize the backend of WordPress to provide a more customized experience for you and your clients or customers. 

CUSTOM ADMIN MENU FEATURES
- Hide/show any menu or menu item.
- Restore a menu to the last save or to the standard WordPress menu.
- Change the login screen background color
- Change the login screen logo
- Change the login screen logo title
- Change the login screen logo url

== Installation ==

1. Download the admin-setting.zip file to your computer.
2. Unzip the file.
3. Upload the admin-setting directory to your /wp-content/plugins/ directory.
4. Activate the plugin through the 'Admin Setting' menu in WordPress.
